# Tower Defense

Игра на Pygame в жанре Tower Defense, вдохновленная Infinitode 2.

## Установка зависимостей

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

## Что есть в игре

- 3 заранее подготовленных уровня и 1 случайный режим.
- 6 типов башен с прокачкой, уровнями и способностями.
- Волны врагов с постепенным ростом сложности.
- Постоянные улучшения между забегами.
- Фоновая музыка из папки `data/music`.

## Управление

- ЛКМ по кнопке справа: выбрать башню для строительства.
- ЛКМ по доступной клетке на карте: построить выбранную башню.
- ЛКМ по уже построенной башне: выбрать ее и открыть информацию справа.
- `Start Wave`: запустить следующую волну.
- `Upgrade`: улучшить выбранную башню за монеты.
- На 4 и 7 уровне башни можно выбрать одну из способностей.
- На 10 уровне недостающая способность выбирается автоматически.
- На 20 уровне башня получает `Ultimate`.
- `Esc` или `P`: пауза.

## Башни

- **Basic**: дешевая стартовая башня по наземным целям.
- **Sniper**: высокая дальность и урон, низкая скорострельность; особенно хороша против бронированных врагов.
- **Cannon**: стреляет медленными взрывными снарядами и наносит урон по области.
- **Freeze**: замедляет наземных врагов в радиусе и наносит им небольшой постоянный урон.
- **Anti-air**: атакует только воздушные цели и делает это быстрее остальных.
- **Splash**: выпускает снаряды во все стороны и хорошо работает против групп врагов.

У всех башен есть:

- внутренний уровень от 1 до 20;
- отдельный уровень улучшения за монеты;
- три базовые способности: `Faster Attack`, `More Damage`, `Longer Range`;
- `Ultimate` на 20 уровне.

## Враги

- **Normal**: базовый противник без особенностей.
- **Fast**: быстрый, но с меньшим запасом здоровья.
- **Strong**: медленный и более живучий.
- **Armored**: снижает входящий урон почти от всех источников, кроме снайпера.
- **Air**: воздушный враг, которого может атаковать только `Anti-air`.
- **Boss**: появляется в последней волне уровня и сопровождается усиленной пачкой других врагов.

С каждой новой волной враги получают больше здоровья, немного ускоряются и дают больше награды.

## Уровни

- **Level 1**: 10 волн, базовая сложность.
- **Level 2**: 15 волн, больше врагов и повышенный множитель сложности.
- **Level 3**: 20 волн, самая плотная из фиксированных карт.
- **Level R**: случайная карта и случайное количество волн.

На последней волне любого уровня появляется босс.

## Прогресс и сохранения

При первом запуске автоматически создается файл `data/save.json`. В нем хранятся:

- количество `meta_coins`;
- уровни постоянных улучшений;
- выбранная скорость игры;
- громкость музыки.

`Meta coins` зарабатываются за убийство врагов и тратятся в меню `Upgrades`.

Доступные постоянные улучшения:

- `Max HP`
- `Tower Damage Bonus`
- `Tower Attack Speed Bonus`
- `Start Coins`
- `Kill Reward Bonus`

В главном меню также есть `Reset Progress`: он сбрасывает улучшения и возвращает потраченные на них meta coins.

## Структура проекта

- `main.py`: запуск Pygame и вход в главное меню.
- `src/menu.py`: главное меню, выбор уровня, постоянные улучшения и сброс прогресса.
- `src/game.py`: игровой цикл, обработка событий, пауза и отрисовка матча.
- `src/game_session.py`: состояние текущего забега, строительство башен и итог матча.
- `src/map.py`: карта, дорожка, клетки для строительства и работа с координатами.
- `src/map_generator.py`: генерация случайной карты для режима `Level R`.
- `src/wave_manager.py`: состав волн, спавн врагов и завершение уровня.
- `src/enemy.py`: характеристики, движение, урон и эффекты врагов.
- `src/tower.py`: логика башен, прокачки, способности и `Ultimate`.
- `src/projectile.py`: движение снарядов и обработка попаданий.
- `src/renderers.py`: отрисовка башен, врагов, радиуса атаки и снарядов.
- `src/player.py`: здоровье игрока, монеты и постоянные улучшения.
- `src/ui.py`: правая панель, кнопки, слайдеры и экран паузы.
- `src/audio.py`: загрузка и воспроизведение музыки из `data/music`.
- `src/utils.py`: сохранение JSON и вспомогательные функции.
- `src/constants.py`: игровые константы, параметры башен, волн и уровней.

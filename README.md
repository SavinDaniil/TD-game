# Tower Defense

Tower Defense на Python, вдохновленный игрой Infinitode 2.

## Установка зависимостей

```bash
pip install -r requirements.txt
```

## Запуск игры

```bash
python main.py
```

## Управление

- ЛКМ по кнопке на панели справа: выбрать тип башни.
- ЛКМ по клетке карты: построить выбранную башню.
- ЛКМ по построенной башне: открыть панель управления башней.
- Кнопки способностей появляются в панели выбранной башни на уровнях 4 и 7.
- `Powerful / Ultimate`: выбрать финальную способность на 20 уровне.
- `Esc`: снять выбор.
- `P`: пауза.

## Башни

- **Basic**: дешевая универсальная стартовая башня по наземным врагам.
- **Sniper**: большой урон и радиус, медленная стрельба, хорош против брони и босса.
- **Cannon**: медленные взрывные снаряды с уроном по области.
- **Freeze**: пассивно замедляет наземных врагов в радиусе.
- **Anti-air**: быстрая башня, только против воздушных врагов.
- **Splash**: выпускает снаряды во все стороны.

## Враги

- **Normal**: обычный враг со средними параметрами.
- **Fast**: быстрый, но слабый.
- **Strong**: медленный и крепкий.
- **Armored**: снижает обычный урон, но уязвим к снайперу.
- **Air**: летит по тому же пути, атакуется только анти-эиром.
- **Brute**: босс последней волны с большим запасом HP.

## Уровни

- **Level 1**: 10 волн.
- **Level 2**: 15 волн, другая дорога и больше врагов.
- **Level 3**: 20 волн, длинная дорога и самые плотные волны.

## Прогресс

Файл `data/save.json` создается автоматически, если отсутствует. В нем хранятся:

- постоянные монеты;
- уровни постоянных улучшений;
- лучшая достигнутая волна.

Постоянные улучшения доступны в меню `Upgrades`: Max HP, Tower Damage Bonus, Tower Attack Speed Bonus, Start Coins и Kill Reward Bonus.

## Структура

- `main.py`: точка входа.
- `src/menu.py`: главное меню и меню постоянных улучшений.
- `src/game.py`: игровой цикл, события, обновление и отрисовка.
- `src/map.py`: сетка, дорога и места для башен.
- `src/player.py`: HP, монеты и постоянные улучшения.
- `src/wave_manager.py`: создание и запуск волн.
- `src/enemy.py`: враги и босс.
- `src/tower.py`: базовый класс башни и 6 типов башен.
- `src/projectile.py`: снаряды и взрывы.
- `src/ui.py`: правая панель, кнопки и информация.
- `src/utils.py`: сохранение JSON и общие функции.
